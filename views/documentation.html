POST /api

Send an array of trips. Trips must be composed of these fields:
  • boo_return:
    • type : true | false
    • description : Idicates whether the route is has a return (true) or is a one-way (false)
  • co2_kg:
    • type : float
    • description : Indicates whether the route has a return (true) or is a one-way (false)
  • price_eur:
    • type : float
    • description : The price of the trip in euro
  • workTime_sec:
    • type : float
    • description : The time duration of the trip permitting you to work
  • duration_out_sec:
    • type : float
    • description : The time duration of the trip
  • desc:
    • type : string
    • description : The description of the trip
  • id:
    • type : string
    • description : The id of the trip
